## 关键概念

- `class` 可以先理解成“能保存状态的函数包”。
- `self` 表示“当前这个对象自己”。
- `__init__` 用于初始化对象，把模型客户端、工具箱、最大步数、历史记录等信息保存下来。
- `run()` 是 Agent 真正开始工作的入口。
- ReAct 的核心流程是：
  1. 把工具列表、用户问题、历史记录拼成 prompt。
  2. 调用大模型生成 Thought 和 Action。
  3. Python 解析 Action。
  4. 如果 Action 是工具调用，就执行工具。
  5. 把工具返回结果作为 Observation 放回历史。
  6. 继续下一轮，直到模型输出 Finish。
