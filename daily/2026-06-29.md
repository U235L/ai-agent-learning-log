## 关键概念

- `self` 表示当前对象本身，不是普通实参。
- `self.xxx` 表示当前对象自己保存的属性。
- `__init__` 会在对象创建时执行，用来给对象保存初始状态。
- 同一个类可以创建多个对象，不同对象的状态彼此独立。
- `class` 的价值是把一组相关的数据和操作打包在一起。

## 今天读懂的代码/结构

以 `ReActAgent` 为例：

```python
class ReActAgent:
    def __init__(self, llm_client, tool_executor, max_steps=5):
        self.llm_client = llm_client
        self.tool_executor = tool_executor
        self.max_steps = max_steps
        self.history = []
```

这段代码的含义是：

- `self.llm_client`：当前 Agent 保存的大模型客户端。
- `self.tool_executor`：当前 Agent 保存的工具箱。
- `self.max_steps`：当前 Agent 最多循环几轮。
- `self.history`：当前 Agent 自己的历史记录。

因此，当执行：

```python
agent = ReActAgent(llm_client, tool_executor)
```

就相当于创建了一个带有大模型、工具箱和历史记录的 Agent 对象。

## 与 ReAct Agent 的关系

ReActAgent 适合用 class，是因为它需要持续保存状态：

- 每一轮都要访问同一个工具箱。
- 每一轮都要把新的 `Action` 和 `Observation` 追加到同一个 `history`。
- 多个方法之间需要共享 `llm_client`、`tool_executor`、`max_steps` 和 `history`。

如果不用 class，就需要在函数之间反复传递这些变量；用 class 后，它们都保存在 `self` 上，代码结构更清楚。
